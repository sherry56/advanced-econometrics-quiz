题型 分数 复习重点

案例分析题复习大纲

面板模型
  • 基准双向固定效应
  • GMM 动态面板
  • 门限面板、分位数面板
  • 空间面板与溢出效应
微观计量模型
  • 二项选择模型
  • 有序 / 无序多项选择
  • 截尾、截断、计数模型
  • 生存分析与久期数据
因果推断模型
  • 工具变量与内生性
  • DID / PSM-DID
  • RDD 断点回归
  • 政策评价与经济意义解释

答题流程

1. 选模型

先判断数据类型、被解释变量类型和题目关键词，再选择对应模型。

2. 写设定式

写出被解释变量、核心解释变量、控制变量、固定效应或动态项，并明确误差项。

3. 写 Stata 命令

主命令要体现模型类型、固定效应、聚类稳健标准误或 GMM 选项。

4. 解释经济意义

围绕核心解释变量说明方向、显著性和幅度，注意被解释变量是否取对数。

答题提醒：卷子尽量写满，不留空白；模型选择理由、设定式、命令和经济意义都写上。若题目没有给具体估计结果，可以直接假设核心系数显著并解释其经济含义。
